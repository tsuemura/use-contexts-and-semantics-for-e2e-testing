# コンテキストとセマンティクスを意識してリーダブルなE2Eテストコードを書こう

[リーダブルなテストコードについて考えよう ～VeriServe Test Automation Talk No.3～](https://veriserve-event.connpass.com/event/243280/) における発表資料です。

## サンプルコードの使い方

事前にNodeJSとnpmコマンドが利用できるようにしておいてください。

```bash
$ node -v
$ npm -v
```

パッケージをインストールします。

```bash
$ cd sample
$ npm install
```

Cypressを起動します。

```bash
$ npx cypress open
```
